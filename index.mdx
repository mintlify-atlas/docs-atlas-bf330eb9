The OpenTelemetry Operator is a Kubernetes Operator that manages the full lifecycle of [OpenTelemetry Collectors](https://github.com/open-telemetry/opentelemetry-collector) and automates the injection of OpenTelemetry instrumentation libraries into your workloads — no code changes required.

## Get started

<Steps>
  <Step title="Install cert-manager">
    The operator requires cert-manager for webhook TLS certificates.
    ```bash
    kubectl apply -f https://github.com/cert-manager/cert-manager/releases/latest/download/cert-manager.yaml
    ```
  </Step>
  <Step title="Install the operator">
    Apply the latest operator release manifest to your cluster.
    ```bash
    kubectl apply -f https://github.com/open-telemetry/opentelemetry-operator/releases/latest/download/opentelemetry-operator.yaml
    ```
  </Step>
  <Step title="Create a Collector">
    Deploy an OpenTelemetry Collector instance using the `OpenTelemetryCollector` CRD.
    ```yaml
    apiVersion: opentelemetry.io/v1beta1
    kind: OpenTelemetryCollector
    metadata:
      name: my-collector
    spec:
      config:
        receivers:
          otlp:
            protocols:
              grpc:
                endpoint: 0.0.0.0:4317
              http:
                endpoint: 0.0.0.0:4318
        exporters:
          debug: {}
        service:
          pipelines:
            traces:
              receivers: [otlp]
              exporters: [debug]
    ```
  </Step>
  <Step title="Enable auto-instrumentation">
    Create an `Instrumentation` resource and annotate your pods to inject tracing automatically.
    ```bash
    instrumentation.opentelemetry.io/inject-java: "true"
    ```
  </Step>
</Steps>

## Supported instrumentation languages

The operator supports automatic instrumentation injection for the following languages:

| Language | Annotation | Default |
|---|---|---|
| Java | `instrumentation.opentelemetry.io/inject-java` | Enabled |
| Node.js | `instrumentation.opentelemetry.io/inject-nodejs` | Enabled |
| Python | `instrumentation.opentelemetry.io/inject-python` | Enabled |
| .NET | `instrumentation.opentelemetry.io/inject-dotnet` | Enabled |
| Apache HTTPD | `instrumentation.opentelemetry.io/inject-apache-httpd` | Enabled |
| Go | `instrumentation.opentelemetry.io/inject-go` | Disabled |
| Nginx | `instrumentation.opentelemetry.io/inject-nginx` | Disabled |
